# Side-Project-Studio-Backstage

* 說明：有 ctrl 為快捷鍵 / 無則是指令

## 終端機指令&快捷鍵

`ctrl + ~` 開啟終端機

`ctrl + c` 終止程序

`cls` 清除屏幕上的所有显示

`cd 位址` 移動終端機至該位址執行

## Node 相關應用

[NPM、NVM](https://ithelp.ithome.com.tw/articles/10230877)



`nvm` 查詢 node 版本管理，執行版本與相關指令

`node -v` 查詢當前使用的 node 版本

`npm ls` 查詢當前目錄安裝了那些套件版本

`npm install npm -g` 更新 npm
